# MCCLIB
MCCLIB is a library for C#.

### Note that this package is still in development.

## Documentation

### Calculator:

Add Method:

```CSharp

using MCCLIB.Commands;

class Sample {
  static void Main(string[] args) {
    Calculator.Add(5, 5, 5);
  }
}

```
